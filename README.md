A modern, high-performance e-commerce storefront template built with [deco.cx](https://deco.cx). This template provides a complete foundation for building fast, scalable online stores with features like product listings, shopping cart, user authentication, wishlists, and more.

## What is this project?

This is a production-ready e-commerce storefront template that leverages deco.cx's edge-first architecture for optimal performance. It comes pre-configured with:

- **Product Management**: Product listings, detail pages (PDP), and category pages (PLP)
- **Shopping Cart**: Full cart functionality with multiple platform integrations (Shopify, VTEX, Wake, Nuvemshop, and more)
- **User Features**: Authentication, wishlists, and user sessions
- **UI Components**: Pre-built components for headers, footers, product galleries, search, and navigation
- **Platform Integrations**: Ready-to-use integrations with popular e-commerce platforms
- **Responsive Design**: Built with Tailwind CSS and DaisyUI for beautiful, mobile-first interfaces
- **Edge-Optimized**: Powered by Deno and Fresh framework for lightning-fast performance

Perfect for developers looking to quickly launch a modern e-commerce site or brands wanting a customizable storefront solution.

## Usage

1 - [Install deno](https://docs.deno.com/runtime/manual/getting_started/installation/)


2 - Run the command

```sh
deno task start
```

This will install all dependencies and start your project.

Access `http://localhost:8000` to see your site.
